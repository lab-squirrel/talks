## Lesson 1

這天要從最最最基礎的 HTML 和 CSS 開始介紹。會先跟大家介紹從瀏覽器打下一個網址之後，網路做的奇妙事情。並且告訴大家每個東西（HTML/CSS/伺服器/瀏覽器）扮演的角色是什麼。當然絕對會有實際寫程式的部份，不寫程式就不是程式課程了。

- 了幾 Domain 和 IP 的關係
- 了解前端、後端、伺服器、資料庫的責任
- 了解 HTML 和 CSS 的功能和 syntax
- 製作簡單的網頁頁面
- 上傳網站到 [GitHub Pages](https://pages.github.com)
